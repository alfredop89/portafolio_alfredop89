Mi nombre es,

... y soy desarrollador web!

Soy de Venezuela, tengo 32 años de edad y vivo en la ciudad de Caracas. Estudié informática, Comunicación social y actualmente soy desarrollador web. La creatividad y el aprendizaje constante en la tecnología es mi fuerte. He tenido la mejor disposición al momento de empezar a desarrollar proyectos y a la realización de mis labores para seguir adquiriendo conocimientos de manera consante y de esta forma ir mejorando a medida se finalizan los mismos.

HTML 5

Maquetación y estructuración del código de la página web mediante el uso de etiquetas, atributos y clases a través del editor de código Visual Studio Code.

CSS

Diseño y personalización de la página web a través del stylesheet en cascada que ofrece CSS por medio de fondos, colores, tiopgrafías, flexbox y grid

Bootstrap

Framework que facilita la maquetación y estructuración de la página web mediante una librería personalizada que ofrece un sin número de opciones a elegir para el desarrollo de la misma

Sass

Este lenguaje de hoja de estilos en cascada simplifica y optimiza la hoja inicial de CSS. Además, permite al desarrollador acortar el código para un resultado más ordenado y sencillo

Git

Esta herramienta se utiliza para establecer y armar un control de versiones con su respectivo mantenimiento para el historial general y seguro de la web en construcción

Github

Te permite crear repositorios para el almacenamiento y guardado del código escrito de la página web directamente en internet

Mis proyectos son elaborados con los requerimientos necesarios para la correcta maquetación y armado de la estructura de una página web. Esto incluye etiquetas semánticas, SEO básico, paleta de colores personalizada, elaboración de logotipo, redacción de textos web y diseño responsive adaptable a todos los dispositivos

Creatus Est fue mi proyecto final para obtener el certificado de desarrollador web en Codehouse Argentina y trata de una empresa ficticia dedicada a la fotografía digital, con el objetivo de alcanzar altos niveles de competencia a través de la venta de imágenes exclusivas realizadas por los principales fotógrafos que trabajan para la marca. Asímismo, ofrece los servicios de almacenamiento y edición de imágenes para artistas que quieran probar la experiencia ofrecida.
